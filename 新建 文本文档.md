Dictionary<Key,Value>
KV存储，具体读的时候要写读的是k还是v，
例如item.Value
item其实包含了k和v，需要具体再访问value才是真正的对象



**`Dictionary` 不是线程安全的，`ConcurrentDictionary` 是线程安全的**。

|            | `Dictionary` | `ConcurrentDictionary`       |
| :--------- | :----------- | ---------------------------- |
| 多线程写入 | ❌ 会崩       | ✅ 安全                       |
| 遍历时删除 | ❌ 抛异常     | ✅ 安全                       |
| 性能       | 快           | 稍慢（有锁开销，但做了优化） |
| 适用场景   | 单线程       | 多线程并发读写               |
